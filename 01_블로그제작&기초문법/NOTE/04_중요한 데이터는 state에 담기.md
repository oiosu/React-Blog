##### 3. About state

> 자료를 잠깐 저장할 땐 변수에 담아 사용해도 되지만, 리액트에서는 `state`를 사용해도 된다. 

(1) import { useState }

```react
import { useState } from 'react';
```

(2) useState(보관할 장소)

```react
let [a, b] = useState('남자 가디건 추천')
```

> useState에 담긴 정보를 사용하고 싶다면 `let [작명,  작명]`  앞에 선언해줘야 한다. 

> `let [a, b]`
>
> * `a` : state에 보관했던 자료 
> * `b` : state 변경 도와주는 함수 
> * 작명은 직관적으로 작성하기

##### ◼ Destructuring 문법 

```javascript
let num = [1, 2];
let a = num[0];
let c = num[1];
```

```javascript
let num = [1, 2];
let [a, c] =[1, 2];
```

> array 안에 있던 자료들을 각각 변수로 빼주는 문법이다.
